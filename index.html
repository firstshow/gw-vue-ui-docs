GW-VUE-UI

轻量、可靠的移动端 Vue 组件库

快速上手 →

代码规范

引入eslint和stylelint,能强化你的代码书写规范，养成良好的代码书写习惯。

文档完善

每个组件都有详细的文档说明，真正做到细致如斯，让你享受这样的极致浏览。

高性能

GW-VUE-UI为每个组件都进行了单元测试，而且测试覆盖率高达90%，确保组件的稳定和高性能。

# 像数 1, 2, 3, 4 一样容易

# 安装
yarn add gw-vue-ui -S
# 或者：
npm install gw-vue-ui -S
# 引入
import gwui from 'gw-vue-ui'
import 'gw-vue-ui/lib/style/index.css'
# 注册
Vue.use(gwui)
# 使用
<gw-button size="small">small</gw-button>
