# 定制方法

# 步骤一 引入样式源文件

定制主题时，需要引入组件对应的 Scss 样式文件，支持按需引入和手动引入两种方式。

# 按需引入样式（推荐）

在 babel.config.js 中配置按需引入样式源文件。

module.exports = {
  plugins: [
    [
      'import',
      {
        libraryName: 'gw-vue-ui',
        libraryDirectory: 'es',
        // 指定样式路径
        style: (name) => `${name}/style/scss`,
      },
      'gw-vue-ui',
    ],
  ],
};
