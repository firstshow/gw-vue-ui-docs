# 基本用法

Layout 组件提供了 24列栅格，通过在 Col 上添加 span 属性设置列所占的宽度百分比。此外，添加 offset 属性可以设置列的偏移宽度，计算方式与 span 相同。

<gw-row>
  <gw-col :span="8">span:8</gw-col>
  <gw-col :span="8">span:8</gw-col>
  <gw-col :span="8">span:8</gw-col>
</gw-row>
<gw-row>
  <gw-col :span="4">span:4</gw-col>
  <gw-col :span="10" :offset="4">span:10, offset:4</gw-col>
</gw-row>
<gw-row>
  <gw-col offset="12" span="12">offset: 12, span: 12</gw-col>
</gw-row>

# 在元素中间添加间距

通过 gutter 属性可以设置列元素之间的间距(像素值)，默认间距为 0

<gw-row gutter="10">
  <gw-col span="8">span: 8</gw-col>
  <gw-col span="8">span: 8</gw-col>
  <gw-col span="8">span: 8</gw-col>
</gw-row>

# Flex 布局

<!-- 居中对齐 -->
<gw-row type="flex" justify="center">
  <gw-col span="6">span: 6</gw-col>
  <gw-col span="6">span: 6</gw-col>
  <gw-col span="6">span: 6</gw-col>
</gw-row>
<!-- 右对齐 -->
<gw-row type="flex" justify="end">
  <gw-col span="6">span: 6</gw-col>
  <gw-col span="6">span: 6</gw-col>
  <gw-col span="6">span: 6</gw-col>
</gw-row>
<!-- 两端对齐 -->
<gw-row type="flex" justify="space-between">
  <gw-col span="6">span: 6</gw-col>
  <gw-col span="6">span: 6</gw-col>
  <gw-col span="6">span: 6</gw-col>
</gw-row>
<!-- 每个元素的两侧间隔相等 -->
<gw-row type="flex" justify="space-around">
  <gw-col span="6">span: 6</gw-col>
  <gw-col span="6">span: 6</gw-col>
  <gw-col span="6">span: 6</gw-col>
</gw-row>

# API

# Row Props

参数说明类型默认值
type布局方式，可选值为 flexString-
justifyFlex 主轴对齐方式，可选值为 `'start''end''center'
alignFlex 交叉轴对齐方式，可选值为`'top''middle''bottom'`
gutter列元素之间的间距（单位为 px）[Number, String]-

# Col Props

参数说明类型默认值
span列元素宽度Number-
offset列元素偏移距离[Number, String]-
