# Cell 单元格

# 介绍

单元格为列表中的单个展示项。

# 引入

import { CellGroup, Cell } from 'gw-vue-ui'

export default {
  components: {
    gwCell: Cell,
    gwCellGroup: CellGroup
  }
}

# 代码演示

# 基础用法

Cell 可以单独使用，也可以与 CellGroup 搭配使用，CellGroup 可以为 Cell 提供上下外边框。

<gw-cell-group>
  <gw-cell title="单元格" value="内容" />
  <gw-cell title="单元格" value="内容" label="描述信息" />
</gw-cell-group>

# 展示图标

通过 icon 属性在标题左侧展示图标。

<gw-cell title="单元格" icon="location-o" />
1

# 展示箭头

设置 is-link 属性后会在单元格右侧显示箭头。

<gw-cell title="单元格" is-link />
<gw-cell title="单元格" is-link value="内容" />
1
2

# 页面导航

可以通过 url 属性进行 URL 跳转，或通过 to 属性进行路由跳转。

<gw-cell title="URL 跳转" is-link url="/fs/mobile.html" />
<gw-cell title="路由跳转" is-link to="index" />
1
2

# 分组标题

通过 CellGroup 的 title 属性可以指定分组标题。

<gw-cell-group title="分组1">
  <gw-cell title="单元格" value="内容" />
</gw-cell-group>
<gw-cell-group title="分组2">
  <gw-cell title="单元格" value="内容" />
</gw-cell-group>

# 使用插槽

如以上用法不能满足你的需求，可以使用插槽来自定义内容。

<gw-cell value="内容" is-link>
  <!-- 使用 title 插槽来自定义标题 -->
  <template slot="title">
    <span class="custom-title">单元格</span>
    <span class="sub-title">备注内容</span>
  </template>
</gw-cell>

<gw-cell title="单元格" icon="shop-o">
  <!-- 使用 right-icon 插槽来自定义右侧图标 -->
  <template slot="right-icon">
    <gw-icon name="search" class="search-icon" />
  </template>
</gw-cell>

<style>
  .custom-title {
    margin-right: 4px;
    vertical-align: middle;
  }

  .sub-title {
    color: #919191;
    font-size: 12px;
  }

  .search-icon {
    font-size: 16px;
    line-height: inherit;
  }
</style>

# 垂直居中

通过 center 属性可以让 Cell 的左右内容都垂直居中。

<gw-cell center title="单元格" value="内容" label="描述信息" />
1

# API

# CellGroup Props

参数说明类型默认值
title分组标题string-
border是否显示外边框booleantrue

# Cell Props

参数说明类型默认值
title左侧标题number | string-
value右侧内容number | string-
label标题下方的描述信息string-
icon左侧图标名称或图片链接string-
url点击后跳转的链接地址string-
to点击后跳转的目标路由对象，同 vue-router 的 to 属性open in new windowstring | object-
border是否显示内边框booleantrue
clickable是否开启点击反馈booleannull
is-link是否展示右侧箭头并开启点击反馈booleanfalse
center是否使内容垂直居中booleanfalse
title-style左侧标题额外样式string | Array | object-
title-class左侧标题额外类名string | Array | object-
value-class右侧内容额外类名string | Array | object-
label-class描述信息额外类名string | Array | object-

# CellGroup Slots

名称说明

# Cell Slots

名称说明
default自定义右侧 value 的内容
title自定义左侧 title 的内容
label自定义标题下方 label 的内容
icon自定义左侧图标
right-icon自定义右侧按钮，默认为arrow
extra自定义单元格最右侧的额外内容
