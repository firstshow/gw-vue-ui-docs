# 显示右侧文案

<div>显示右侧文案</div>
<gw-search v-model:value="text3" :isShowOperation="'true'" @handleAction="search" @clearAction="clearAction"></gw-search>
<gw-search v-model:value="text4" cancelText="搜索" actionType="search" :isShowOperation="'true'"  @handleAction="realSearch" @clearAction="clearAction"></gw-search>

<script lang="ts">
import { Search } from 'gw-vue-ui'

export default {
  components: {
    GwSearch: Search
  },
  data () {
    return {
      text: '',
      searchText: '',
      text1: '',
      text2: '',
      text3: 'text3',
      text4: ''
    }
  },
  methods: {
    search (val: string) {
      console.log(val, '搜索')
    },
    clearAction () {
      (this as any).text = '';
      (this as any).text1 = '';
      (this as any).text2 = '';
      (this as any).text3 = '';
      (this as any).text4 = '';
      (this as any).searchText = ''
    },
    realSearch () {
      console.log('执行搜索')
    },
    getFocus () {
      ((this as any).$refs['search'] as any).getFocus()
    }
  }
}
</script>

# Props

参数说明类型值必填
placeholderTextplaceholder 内容String默认 search否
cancelText取消文案String默认 取消否
clearable是否显示清除按钮Boolean默认 true否
type输入框类型String默认 text否
maxLength输入内容限制Number默认 1000否
actionType右侧按钮类型String默认 cancel， 可选择 search否
isAutoFocus是否自动聚焦Boolean默认 false否
isShowOperation是否一直显示右侧操作按钮Boolean默认 false否
